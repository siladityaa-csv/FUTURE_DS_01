***Business Sales Performance Analysis***



1\.**Project Overview**

This project analyzes retail sales data (9,000+ transactions) to identify revenue trends, top-performing products, profitable categories, and regional performance insights.



2\.**Tools Used**

* Python (Pandas, Matplotlib)
* Power BI
* DAX
* Data Cleaning \& KPI Analysis



3\.**Key Insights**

* Technology category generates highest profit.
* Furniture shows high sales but lower margins.
* West region leads in profitability.
* Sales show strong seasonal growth patterns.
